# GPU Matmul Lab

**Status:** Planned (Milestone 4)

Compare matrix multiplication across: Python loops, NumPy, PyTorch CPU, PyTorch GPU,
CUDA kernels (naive then tiled), and different numeric precisions. Measure latency,
throughput, and memory.

## Prerequisites

Complete the CUDA intro path before custom kernels:

1. [An Even Easier Introduction to CUDA](https://developer.nvidia.com/blog/even-easier-introduction-cuda/)
2. [CUDA Programming Guide](https://docs.nvidia.com/cuda/cuda-programming-guide/index.html) — Ch. on thread hierarchy and memory
3. [Official CUDA exercises (PDF)](https://www.nvidia.com/content/cudazone/download/Exercise_Instructions.pdf) via [exercises/cuda/](../../exercises/cuda/)
