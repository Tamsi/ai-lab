# Roadmap

This document tracks the major learning milestones. Each milestone has clear
objectives and a capstone project. See [GitHub milestones](https://github.com/Tamsi/ai-lab/milestones) for issue tracking.

---

## Milestone 1: Mathematical foundations

**Status:** In progress

### Objectives

- [ ] Functions and logarithms
- [ ] Vectors and matrices
- [ ] Derivatives
- [ ] Probability
- [ ] Statistics
- [ ] Optimization

### Capstone project

Implement and visualize linear regression and logistic regression.

### Week 1–4 plan (first month)

| Week | Topics | Deliverables |
|---|---|---|
| 1 | Functions, vectors, sigma notation, dot product, matrix multiplication | 2 notes, 2 exercises, benchmarks |
| 2 | Linear transformations, norms, distances, vector visualization | 2 notes, Transformer intro paper sheet |
| 3 | Derivatives, chain rule, gradients | Start Gradient Playground |
| 4 | Probability, expectation, variance, softmax, cross-entropy | Monthly review in PROGRESS.md |
