# AI Lab

Personal notebook for learning AI: courses, concepts, notes, exercises.

## Four places

| File / folder | What goes here |
|---|---|
| [courses.md](courses.md) | Courses you plan to follow (links + status) |
| [concepts.md](concepts.md) | Notions you discover, with links to notes |
| [notes/](notes/) | Your notes — by course or by concept |
| [exercises/](exercises/) | Exercise subjects + where you solve them |

## Quick links

- Course notes → `notes/courses/<course-name>.md`
- Concept notes → `notes/concepts/<concept-name>.md`
- Solve an exercise → `exercises/<subject>/solution.py`

## Verify an exercise

```bash
pip install numpy pytest
pytest exercises/dot-product
```
