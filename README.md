# AI Lab

Personal notebook for learning AI: courses, papers, concepts, notes, exercises.

## Five places

| File / folder | What goes here |
|---|---|
| [courses.md](courses.md) | Courses you plan to follow |
| [papers/](papers/) | Papers you read + one note per paper |
| [concepts.md](concepts.md) | Notions you discover |
| [notes/](notes/) | Deeper notes by course or concept |
| [exercises/](exercises/) | Exercise subjects + where you solve them |

## Quick links

- Course notes → `notes/courses/<name>.md`
- Paper notes → `papers/<arxiv-id>-<slug>.md`
- Concept notes → `notes/concepts/<name>.md`
- Solve an exercise → `exercises/<subject>/solution.py`

## Verify an exercise

```bash
pip install numpy pytest
pytest exercises/dot-product
```
